Mycelium Entropy Shared Secret Reassembly

If you have private key shares generated by a Mycelium Entropy device in Shamir’s Secret Sharing mode, this tool allows you to combine them and obtain the private key.

Please bear in mind that although this tool runs entirely in your browser, there is no guarantee that the recovered private key remains confidential. Please treat it as if it may have been compromised, and move the funds to a safe place as quickly as possible.

This tool does not intentionally transmit or store your data.
